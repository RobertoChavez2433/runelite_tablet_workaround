# Research Artifacts

Research gathered during brainstorming and planning phases for the Runelite for Tablet project.

## Files

| File | Description |
|------|-------------|
| `jagex-launcher-auth.md` | Jagex Launcher authentication flow and RuneLite integration |
| `runelite-architecture.md` | RuneLite technical architecture, dependencies, rendering |
| `android-approaches.md` | Methods for running Java desktop apps on Android |
| `existing-projects.md` | GitHub repos and community attempts at RuneLite on Android |
| `feasibility-analysis.md` | Synthesized feasibility analysis across all approaches |

## Usage

These files are populated as research agents return findings. They persist across sessions to avoid re-researching the same topics.
